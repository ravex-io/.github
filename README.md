<h1 align="center">Ravex</h1>

<br />

Ravex is an open source chat system built on top of [socket.io](https://socket.io). It provides everything needed to add real-time chat to any Node.js server — rooms, messaging, presence tracking, typing indicators, reactions, and rate limiting — without building from scratch.
